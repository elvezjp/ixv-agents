# PP-DocLayout-L（ビジョンモデル概要）

## 目的
- 文書画像からレイアウト領域（図・表・本文・見出しなど）を検出する。
- ページ単位の重要度判定と可視化の前処理として使用する。

## 位置づけ
- 本システムでは「レイアウト検出」と「可視化生成」に利用する。
- 文字認識（OCR）は含まない前提。

## 入力/出力（概念）
- 入力: ページ画像（PDFをページ画像化したもの）
- 出力: レイアウト領域の矩形（カテゴリ、座標、信頼度）

## 運用メモ
- 中〜高品質の文書画像を想定。
- 重要ページ抽出の判定ロジックは別途スコアリングで実施。
